# sistema-agendamento-consultas
Sistema de agendamento de consultas usando MongoDB e Express.
Esse projeto é uma adaptação de um dos projetos da Formação NodeJS.

## Tecnologias usadas
- NodeJS
- Express
- MongoDB/Mongoose
- FullCalendar
- EJS
